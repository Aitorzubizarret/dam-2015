HTML (HyperText Markup Language) es un lenguaje de marcas usado, de manera predominante, para la elaboracián de páginas web.

< tag atributo1 = "valor1" atributo2 = "valor2" ...> contenido < / tag > Tipos de atributos: - básicos - id="texto" => Indicador ÚNICO para cada elemento - class="texto" => Clase CSS para aplicarle el estilo - title="texto" => Título del elemento (mejora la accesibilidad) - internalización - lang="código" => Idioma del elemento (web) - dir => Dirección del texto - eventos - onclick - ondbclick - onmousedown - onmouseup - onmousemove - onmouseout - onmouseover - onload - onunload - onresize - onkeydown - onkeyup - onkeypress - onblur - onfocus - onchange - onreset - onselect - onsubmit - foco - acceskey="letra" => Tecla de acceso rápido a un elemento HTML - tabindex="número" => Posición del elemento en el orden de tabulación de la web (valor entre 0 y 32.767) - onfocus, onblur => Controlan los eventos JavaScript que se ejecutan cuando el elemento obtiene o pierde el foco.
